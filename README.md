# Canton
一个来自Focusorder的PHP建站项目，项目名称Canton意义为开发团队所在地广东省“粤”的英文名称，它的中文项目名称是“粤”。
# 目录结构说明
##前台目录结构
- 前台HTML文件放在Index/Home/View/目录下
- 前台使用的JS、images和css存放在Public/目录下
##后台目录
- index.php是前台入口文件
- admin.php是后台入口文件
- Index目录是前台系统
- Admin目录是后台系统
- 控制器文件放在 Home/Controller 目录下
- 封装的类放在 ThinkPHP\Library\Think 目录下
- 日志文件生成在 Runtime\Logs 目录下
